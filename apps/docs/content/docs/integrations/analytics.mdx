# Integration: Analytics

Partha will expose opinionated analytics hooks so operators understand activation, retention, and delivery throughput without exporting spreadsheets.

## Targets

- Product analytics providers (e.g. PostHog, Plausible) behind a thin abstraction.
- Event taxonomy covering workspace onboarding, issue throughput, MCP tool usage (aggregated + privacy-safe).

## Principles

- Opt-in per workspace for external forwarding.
- Aggregate MCP telemetry — avoid leaking issue titles to third parties without explicit consent.
