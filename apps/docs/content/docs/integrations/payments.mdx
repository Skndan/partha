# Integration: Payments (Stripe & Razorpay)

Goal: connect subscription lifecycle with workspace milestones so billing signals influence prioritization.

## Scope ideas

- Stripe Billing + Customer Portal for global audiences.

- Unified webhook normalization layer mapping provider events → Partha `activity_event` records.

## Compliance

- Never store raw PAN/card data — rely on provider vaults.
- Clearly separate PCI scope vs configuration metadata stored in Partha.
